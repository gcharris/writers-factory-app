# Gold Standard Notebook Setup Guide

**Purpose:** Create a streamlined NotebookLM notebook optimized for generating high-quality scaffolds for Volumes 2 & 3 of The Explants Trilogy.

**Strategy:** Retire the massive legacy notebook in favor of an Act-based, consolidated architecture that maintains consistency without clutter.

---

## Core Principles

### Why Streamline?

**Problem with Legacy Notebook:**
- ❌ Massive files with historical context clutter
- ❌ Difficult for NotebookLM to prioritize relevant information
- ❌ Inconsistent quality in scaffold generation
- ❌ No clear structure for Act-based development

**Benefits of Gold Standard Notebook:**
- ✅ **Consistency Guarantee:** Core Architecture ensures all chapters adhere to established rules
- ✅ **Act-Based Focus:** Prevents narrative density, maintains pacing
- ✅ **Gold Standard Templates:** ACE prompt automates scaffold expansion
- ✅ **Cognitive Filter:** Only relevant, actionable information prioritized

---

## Recommended Notebook Structure

### Organizational Approach

**Four Core Categories:**
1. Core Architecture & Voice Standards (The Rulebook)
2. Volume Two - Consciousness War (Acts IV, V, VI)

4. Volume One - Continuity Reference

**Implementation:**
- Use folders/tabs in NotebookLM to separate categories
- Within each category, upload specific documents
- Maintain Act-based organization for Volumes 2 & 3

## Migration from Legacy Notebook

### If You Have Existing Massive Notebook

**Don't migrate everything!** Use this strategic filter:

**Keep:**
- ✅ Core architecture docs
- ✅ Completed chapter finals (not drafts)
- ✅ Character profiles (latest versions only)
- ✅ Volume 1 complete manuscript

**Discard:**
- ❌ Historical drafts and iterations
- ❌ Obsolete planning documents
- ❌ Redundant character notes
- ❌ Old scaffolds (replaced by finals)

**Process:**
1. Create new Gold Standard Notebook
2. Upload only essential docs using structure above
3. Test with sample queries
4. Keep legacy notebook as archive (don't delete)
5. Switch to Gold Standard for all new work

## Troubleshooting

### Issue: Scaffolds Missing Strategic Context

**Symptom:** Generated scaffolds lack Vector positioning, Techno-Feudal terminology

**Solution:**
- Verify TRILOGY_IDEOLOGICAL_ARCHITECTURE.md is uploaded
- Check that file uses precise terminology (not generic descriptions)
- Test query: "What is Vector Beta's approach?" Should return Vance/Individual Optimization details

### Issue: No Continuity Callbacks

**Symptom:** Scaffolds don't reference previous chapters or Volume 1

**Solution:**
- Verify THE_EXPLANTS_VOLUME_ONE_COMPLETE.md is uploaded
- Check that completed Volume 2 chapters are uploaded (not just scaffolds)
- Explicitly request continuity: "Include callbacks to Volume 1 Sadie abandonment"

### Issue: Voice Requirements Too Generic

**Symptom:** Scaffolds say "use Enhanced Mickey voice" without specific guidance

**Solution:**
- Verify MICKEY_BARDOT_VOICE_GOLD_STANDARD.md contains specific examples
- Test query: "What metaphors should Mickey use?" Should return gambling/con artistry/performance
- Update voice doc with concrete anti-patterns list
