# Archive

This folder contains historical documentation that has been superseded by current specifications or is no longer actively maintained.

## Subfolders

### /drafts
Draft documents that were never finalized or were superseded during development.

### /superseded
Complete documents that have been replaced by newer versions or consolidated into other specifications.

## Archival Policy

Documents are moved to archive when:
1. **Superseded**: A newer version or consolidated document replaces it
2. **Draft Abandoned**: Draft work that was completed differently than planned
3. **Historical Reference**: Old architecture versions retained for reference
